# Place

- [Live Preview](https://places-92.netlify.app)


## Technologies 

The following technologies were used in this project:

- [HTML](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/HTML_basics/)
- [CSS](https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps)
- [React](https://reactjs.org/)


## Requirements

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed. Alternatively, you can download the code as a zip file.

## Clone this project

    git clone https://github.com/benidevo/places.git

## Install dependencies

    npm install

## Start server

    npm start
